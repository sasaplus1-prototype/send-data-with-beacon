# send-data-with-beacon

send data with Beacon API

## Setup

install dependencies.

```sh
$ npm install
```

start server.

```sh
$ npm start
```

server starting at `http://127.0.0.1:3000/`.

## Libraries

- [body-parser](https://github.com/expressjs/body-parser)
- [Express](http://expressjs.com/)
